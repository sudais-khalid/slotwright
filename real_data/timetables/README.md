# Semester timetables (1–8)

Upload the weekly class timetable of each BSAI semester here.

Accepted: the official Excel/PDF timetable, a photo/screenshot of it, or the
CSV template below. Name files so the semester is obvious, e.g.
`BSAI-1.xlsx`, `semester_3.pdf`, `bsai_5_timetable.png`.

## CSV template (`timetable_template.csv`)

One row per class meeting:

| column | meaning | example |
|---|---|---|
| semester | 1–8 | 6 |
| section | section letter, or `A` if only one | A |
| day | Mon / Tue / Wed / Thu / Fri | Wed |
| start | class start, 24h | 08:30 |
| end | class end, 24h | 10:00 |
| course | course name or code | Reinforcement Learning |
| room | where it's held | Lab-3 |

If a class repeats on several days, write one row per day.
